# youranAccount
悠然账目web应用2018-10-9  

1. 在GitHub上建一个repository  
2. 本地项目右键, Team , share Project , Git , 创建本地repository(勾选use or create)  
3. 新建的本地仓库有 **NO-HEAD** 标识，要先把远程仓库pull到本地仓库，使用https协议  
4. commit 到本地仓库，然后push 到GitHub远程仓库
